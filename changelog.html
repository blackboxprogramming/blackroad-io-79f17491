v0.9.2 January 20, 2025 Latest

Performance improvements and bug fixes. This release focuses on reducing memory usage during inference and improving startup times.

Added

  • Support for Phi-3 model family including phi-3-mini and phi-3-medium
  • New --low-memory flag for memory-constrained devices
  • ARM64 native builds for Raspberry Pi 5
  • Experimental Vulkan backend for AMD GPUs

Changed

  • Reduced memory footprint by 15% during model loading
  • Improved startup time by lazy-loading model metadata
  • Updated llama.cpp to latest upstream (b1842)

Fixed

  • Fixed crash when using CUDA with certain GGUF quantizations
  • Resolved memory leak in long-running inference sessions
  • Fixed incorrect token counting for multi-byte UTF-8 characters

v0.9.1 December 15, 2024 Security

Security patch addressing a vulnerability in the API server. All users are recommended to upgrade.

Security

  • Fixed path traversal vulnerability in model file loading (CVE-2024-XXXXX)
  • Added input validation for API endpoints

Fixed

  • Fixed race condition in concurrent inference requests
  • Resolved issue with Metal backend on macOS Sequoia
v0.9.0 November 28, 2024

Major release introducing OpenAI-compatible API, plugin system, and significant performance improvements.

Added

  • OpenAI-compatible REST API (/v1/chat/completions, /v1/embeddings)
  • Plugin system for extending functionality
  • Support for GGUF v3 format
  • New web UI for interactive chat sessions
  • Model quantization tool (blackroad quantize)
  • Configuration profiles for different use cases

Changed

  • Rewrote inference engine for 2x throughput improvement
  • New configuration file format (automatic migration provided)
  • Minimum macOS version is now 12.0 (Monterey)

Removed

  • Deprecated GGML format support (use GGUF instead)
  • Removed legacy --threads flag (use --n-threads)
v0.8.5 October 10, 2024

Bug fixes and stability improvements.

Fixed

  • Fixed incorrect logits on certain Llama 2 models
  • Resolved Windows installer PATH issues
  • Fixed crash on systems with AVX but no AVX2
v0.8.0 September 1, 2024

Initial public beta release of BlackRoad.

Added

  • Core inference engine with CPU and CUDA support
  • Model management and downloading
  • CLI interface with interactive mode
  • Configuration system
  • Basic documentation
