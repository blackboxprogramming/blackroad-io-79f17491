Expandable Cards · FAQ
What is trinary logic? ▼
Trinary logic extends binary true/false with a third UNKNOWN state, allowing agents to explicitly represent uncertainty. Each state includes a confidence score from 0.0 to 1.0, enabling nuanced reasoning about partial knowledge.
How does contradiction quarantine work? ▼
When the system detects contradictory claims, it isolates them in a quarantine zone using paraconsistent logic. Claims remain accessible but flagged, preventing them from corrupting other reasoning until human review resolves the conflict.
What is PS-SHA∞ hashing? ▼
PS-SHA∞ is a cryptographic hashing algorithm designed for memory persistence. It generates unique identifiers for journal entries, enabling truth state verification and tamper-evident logging of agent memory states.
