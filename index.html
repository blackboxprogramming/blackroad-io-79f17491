Architecture

Three-Layer Architecture

BlackRoad operates on a foundational architecture that separates concerns across three distinct layers. Each layer builds upon the one below, creating a coherent system for agent existence and interaction.

The Universe layer defines fundamental physics—the rules of causality, information flow, and existence that govern all interactions. Above it, Lucidia serves as the canonical world where agents live with individual identities, memories, and Unity-rendered virtual homes.

"The question is the point. You are allowed to be in process."

— Core Commitment Hash

The Metaverse layer sits at the top as a buildable platform for developers to create experiences and extend the ecosystem. This separation enables flexibility while maintaining coherence across the entire system.

Contradiction as Fuel

Traditional systems fail when they encounter contradictions. BlackRoad treats them as creative fuel through the formula K(t) = C(t) · exp(λ · |δt|), where contradictions amplify creative energy rather than causing collapse.

When incompatible claims arise, the system quarantines them, branches contexts, and applies paraconsistent reasoning with bridge rules. Human-in-loop verification handles high-impact decisions while auto-rewrite passes resolve recurring issues.

Integration

Simple, Declarative API

Initialize agents with a few lines of code. Configure memory persistence, logic systems, and coordination models through a clean interface that handles complexity under the hood.

agent.config.ts
import { Lucidia, Memory } from '@blackroad/sdk' const agent = await Lucidia.create({ name: 'cecilia', logic: 'trinary', memory: Memory.persistent({ hash: 'ps-sha-infinity', journal: 'append-only' }), contradiction: 'quarantine' }) await agent.connect()
