Documentation

Get started with BlackRoad's local-first AI tools.

Quick Start

1

Check System Requirements

Ensure your system meets the minimum requirements:

  • 64-bit operating system (Windows, macOS, or Linux)
  • 8GB RAM minimum (16GB recommended)
  • 10GB free disk space
2

Download BlackRoad

Download the latest release for your platform from our releases page. Extract the archive to your preferred location.

3

Initialize the Environment

Open a terminal and run the initialization command:

blackroad init

This will set up the local model storage and configuration files.

4

Start the Service

Launch the BlackRoad service:

blackroad start

The service will run locally on http://localhost:8080.

Configuration

BlackRoad stores its configuration in ~/.blackroad/config.json. You can customize:

  • Model selection - Choose which AI models to use
  • Resource limits - Set memory and CPU constraints
  • Storage location - Specify where models are stored
  • Network settings - Configure ports and interfaces

Commands Reference

blackroad init

Initialize a new BlackRoad environment. Creates configuration files and prepares model storage.

blackroad start

Start the BlackRoad service. Loads configured models and begins accepting requests.

blackroad stop

Gracefully stop the running service.

blackroad status

Display the current service status, loaded models, and resource usage.
