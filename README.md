# BlackRoad

Building sovereign, local-first AI systems.

## Vision

BlackRoad is dedicated to creating AI infrastructure that respects user autonomy and data sovereignty. Our systems are designed to run locally, giving you full control over your AI interactions without relying on external services.

## Status

**Current Phase:** Foundation

The project is actively under development. We're building the core infrastructure for local-first AI systems.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/blackboxprogramming/blackroad-io.git
cd blackroad-io

# Install dependencies
npm install

# Run locally
npm start
```

### Development

```bash
# Run linting
npm run lint

# Format code
npm run format

# Check formatting
npm run format:check
```
