# BlackRoad

Building sovereign, local-first AI systems.

## About

BlackRoad is focused on creating AI infrastructure that prioritizes:

- **User Sovereignty** - Full control over your AI systems and data
- **Local-First** - Core functionality works offline
- **Privacy** - Your data stays on your device

## Status

This project is in early development.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/blackboxprogramming/blackroad-io.git
cd blackroad-io

# Serve locally
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.
